This is where programming begins:
1) Main subject

Introduction to C programming: compiling, basic program structure, printing text, and following Betty coding style.

2) Important points

    C is compiled with gcc; you learn what happens when compiling, assembling, and linking.

    A program starts at main, which is the entry point and affects the return value.

    Output functions include printf, puts, and putchar.

    sizeof is used to get the size of a type.

    You must respect the required compilation flags and Betty style.

    Some tasks are shell scripts that create object files, assembly files, or executables.

    Other tasks are C programs that print exact text and return 0.
