1) Main subject

You’re learning the basics of C: how to compile files, create executables, print text, and understand simple program structure.

2) Important points

    gcc compiles C files; some tasks compile only, others link into an executable.

    main is the entry point of a C program.

    puts prints a string with a newline; printf gives more control; sizeof measures type size.

    You must follow the required compilation flags exactly.

    Betty style matters: clean formatting, comments, and code structure.

    Some tasks are shell scripts, so the file format and executable permissions matter too.

    The output must match the examples exactly.
